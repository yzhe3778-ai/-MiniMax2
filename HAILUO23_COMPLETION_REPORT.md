### 4. 🔌 API 集成

#### 已创建的 API 路由

**生成视频**:
- 路径: `/api/hailuo23/generate`
- 方法: POST
- 功能: 调用 MiniMax API 创建视频生成任务

**查询状态**:
- 路径: `/api/hailuo23/status`
- 方法: GET
- 功能: 查询视频生成任务状态和结果

#### API 调用流程
```
用户提交 → Next.js API → MiniMax API → 返回 task_id
                ↓
        前端轮询状态 ← API 查询
                ↓
        获取视频 URL → 播放/下载
```

## 🔧 配置说明

### 环境变量配置

在 `.env.local` 中添加:

```env
MINIMAX_API_KEY=your-minimax-api-key
```

### 获取 API Key

1. 访问 [MiniMax 平台](https://platform.minimaxi.com/)
2. 注册/登录账号
3. 前往用户中心 → 基本信息 → 接口密钥
4. 创建或复制 API Key
